# MVPFrame

一个Android MVP框架演示项目。

# 目录介绍

- `--base` Activity & Fragment Base类
- `--mvp` MVP的Base类都在这里面
- `--presentation` 主要包，其中的Contract类是定义View和Presenter接口的地方
- `--util` 共通工具包
- `App` Application
- `AppRouter` app路由，界面跳转帮助类，所有的界面跳转通过此类进行跳转,包括组件交互
